<h1>MoveEasy Packers & Movers Landing Page</h1>

<p>
A responsive and modern landing page designed for a Packers & Movers company.
The website focuses on clean UI/UX, mobile responsiveness, smooth animations,
and an SEO-friendly structure.
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
    <li>Responsive Design</li>
    <li>Sticky Navigation Bar</li>
    <li>Hero Section with CTA Buttons</li>
    <li>Services Section</li>
    <li>Why Choose Us Section</li>
    <li>Contact Form</li>
    <li>Smooth Scrolling</li>
    <li>Hover Animations</li>
    <li>Mobile Friendly Layout</li>
    <li>SEO-Friendly Structure</li>
</ul>
